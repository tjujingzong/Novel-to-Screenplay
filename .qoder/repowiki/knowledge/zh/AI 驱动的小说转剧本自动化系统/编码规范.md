- 服务层函数普遍采用异步定义 (`async def`) 以配合 FastAPI 的异步运行时，特别是在涉及 LLM 网络请求的 `converter.py` 和 `llm_client.py` 中。
- 所有业务数据模型均继承自 `pydantic.BaseModel`，利用 Field 描述和类型提示实现自动化的数据校验与 JSON Schema 生成。
- LLM 交互采用统一的 `DeepSeekClient` 封装，支持传入 `response_model` 参数自动完成 JSON 到 Pydantic 模型的解析与清洗。
- 转换过程中的阶段性状态通过 `ConversionStage` 枚举定义，并在 `routes.py` 中通过 `_update_status` 统一更新内存中的任务状态。
