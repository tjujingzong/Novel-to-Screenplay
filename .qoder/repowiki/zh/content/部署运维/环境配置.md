## 简介

小说转剧本工具是一个基于 Python 3.10+ 的 AI 驱动工具，能够将小说文本自动转换为结构化的 YAML 剧本格式。该工具使用 FastAPI 作为后端框架，支持多种文件格式的输入（TXT、Markdown、DOCX、PDF），并通过 DeepSeek API 提供智能的 LLM 驱动转换功能。

## 系统要求

### 最低系统要求

- **Python 版本**: Python 3.10 或更高版本
- **操作系统**: Linux、macOS、Windows
- **内存**: 至少 4GB RAM（建议 8GB+）
- **存储空间**: 至少 500MB 可用空间
- **网络连接**: 需要访问 DeepSeek API

### 硬件推荐规格

- **CPU**: 多核处理器（至少 2 核）
- **内存**: 8GB RAM 或更高
- **存储**: SSD 存储以提高文件处理性能

## Python 环境安装

### 安装 Python 3.10+

#### Windows 系统

1. **从官方网站下载**
   - 访问 [python.org](https://www.python.org/downloads/)
   - 下载 Python 3.10.x 或更高版本的 Windows 安装程序

2. **验证安装**
   ```bash

   pip --version
   ```

#### macOS 系统

1. **使用 Homebrew（推荐）**
   ```bash
   brew install python@3.10
   ```

2. **使用官方安装包**
   - 从 [python.org](https://www.python.org/downloads/macos/) 下载
   - 按照安装向导完成安装

3. **验证安装**
   ```bash
   python3 --version
   pip3 --version
   ```

#### Linux 系统

1. **Ubuntu/Debian**
   ```bash
   sudo apt update
   sudo apt install python3.10 python3.10-venv python3.10-dev
   ```

2. **CentOS/RHEL/Fedora**
   ```bash
   sudo dnf install python3.10 python3.10-venv python3.10-devel
   ```

3. **验证安装**
   ```bash
   python3.10 --version
   pip3.10 --version
   ```

## 环境变量配置

### 配置文件准备

1. **复制示例配置**
   ```bash
   cp .env.example .env
   ```

2. **编辑 .env 文件**
   ```env
   DEEPSEEK_API_KEY=sk-your-key-here
   DEEPSEEK_BASE_URL=https://api.deepseek.com
   DEEPSEEK_MODEL=deepseek-chat
   MAX_UPLOAD_SIZE_MB=50
   DATA_DIR=./data
   ```

### 环境变量详细说明

| 环境变量 | 默认值 | 必填 | 说明 |
|---------|--------|------|------|
| `DEEPSEEK_API_KEY` | 空字符串 | ✅ 是 | DeepSeek API 密钥 |
| `DEEPSEEK_BASE_URL` | `https://api.deepseek.com` | ❌ 否 | API 基础 URL |
| `DEEPSEEK_MODEL` | `deepseek-chat` | ❌ 否 | 使用的模型名称 |
| `MAX_UPLOAD_SIZE_MB` | `50` | ❌ 否 | 最大上传文件大小（MB） |
| `DATA_DIR` | `./data` | ❌ 否 | 运行时数据目录 |

### 高级配置选项

### 生产环境配置

适用于部署和生产使用：

```env

DEBUG=false
LOG_LEVEL=INFO
RELOAD=false

# 安全配置
ALLOW_ORIGINS=https://yourdomain.com
SECRET_KEY=your-secret-key-here

# 性能配置
WORKERS=4
BACKLOG=2048
```

### 环境切换脚本

```bash
#!/bin/bash
# start.sh

case "$1" in
    dev)
        export ENVIRONMENT=development
        export DEBUG=true
        export RELOAD=true
        ;;
    prod)
        export ENVIRONMENT=production
        export DEBUG=false
        export RELOAD=false
        ;;
    *)
        echo "Usage: $0 {dev|prod}"
        exit 1
        ;;
esac

echo "Starting in $ENVIRONMENT mode"
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

## 常见安装问题解决

### Python 版本问题

**问题**: Python 版本过低
```bash
# 错误信息
"RuntimeError: Python version 3.8.10 is too old"
```

**解决方案**:
1. 升级 Python 到 3.10+
2. 更新 PATH 环境变量
3. 验证安装: `python --version`

### 权限问题

**问题**: 权限不足导致安装失败
```bash
# 错误信息
"PermissionError: [Errno 13] Permission denied"
```

**解决方案**:
```bash
# 使用用户目录安装
pip install --user fastapi

# 或者使用虚拟环境
python -m venv ~/.venvs/novel
source ~/.venvs/novel/bin/activate
```

### 依赖冲突

**问题**: 依赖版本冲突
```bash
# 解决方案
pip install --upgrade pip setuptools wheel
pip cache purge
pip install -e ".[dev]"
```

### 编译错误

**问题**: C 编译器错误
```bash
# Linux 解决方案
sudo apt install build-essential libffi-dev libssl-dev

# macOS 解决方案  
xcode-select --install
```

### 网络连接问题

**问题**: 无法连接到 API
```bash
# 检查网络
ping api.deepseek.com

# 配置代理
export HTTP_PROXY=http://proxy:port
export HTTPS_PROXY=https://proxy:port
```

### 端口占用

**问题**: 端口被占用
```bash
# 查找占用端口的进程
lsof -i :8000

# 更换端口
uvicorn app.main:app --port 8001
```
